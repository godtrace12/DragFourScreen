# DragFourScreen
ViewDragHelper Scroller实现自定义布局
1、支持4个View2行2列排列，也可以支持一个View最大化显示。
2、在2x2排列时，可以进行拖动，交换2个View的位置。
3、可以支持拖动来删除(达到删除状态执行自定义的操作)。
4、在放大状态，能通过左右滑动进行前后View的切换。
其中涉及到一些如自定义布局、view拖动、滑动等技术。
